

For at anvende programmeringsproget python skal det installeres fra deres hjemmeside.
Dette gøres på følgende måde: 

1.
gå til https://www.python.org/downloads/ 
2.
klik derefter på den gule knap "Download Python" 
3.
gennemgå installeringen

Det nødvendigt at udføre følgende kommandoer i terminalen for at anvende programmet. 
-	pip install tensorflow
-	pip install keras
-	pip install numpy

Herefter kan programmet startes ved at skrive ”python main.py” i terminalen. 
Koden er blevet dokumenteret i selve programmet, hvor de 7 trin (se afsnit 4) er blevet fremhævet og forklaret.
Det er blandt andet muligt at justere på modellens struktur i filen Main.py
